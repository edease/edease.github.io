Class: QuadrantModel

QuadrantModel(Name, xDimension, yDimension, Q1label, Q2label, Q3label, Q4label, QText)

The GPI Quadrant model is a composite model based on an individual's GPI dimensions

Constructor

new QuadrantModel(Name, xDimension, yDimension, Q1label, Q2label, Q3label, Q4label, QText)

Creates a GPI Quadrant Model
Parameters:
Name Type Description
Name string The name of the Quadrant Model
xDimension Dimension The Dimension associated with the x-axis
yDimension Dimension The Dimension associated with the y-axis
Q1label string Label for the 1st (upper right) quadrant
Q2label string Label for the 2nd (upper left) quadrant
Q3label string Label for the 3rd (lower left) quadrant
Q4label string Label for the 4th (lower right) quadrant
QText string The text associated with this Quadrant Model
